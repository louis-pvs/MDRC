### Usage with Icon
<Playground>
  <Button><Button.Icon iconName="whatshot" />Material Icon</Button>

</Playground>

### Anchor
Pass anchor tag into `htmlTag` props to use it as Anchor
<Playground>
  <Button htmlTag='a' href="https://github.com/louis-pvs/mrcw" target="_blank">
    <GithubIcon className="mdc-button__icon" />Github
  </Button>
</Playground>
